# 🧠 DSA Toolkit in C

A modular implementation of core Data Structures in C using pointers and dynamic memory allocation.

---

## 🚀 Features

### 📦 Data Structures Implemented
- Stack (Array-based)
- Queue (Array-based)
- Linked List (Insertion, Deletion, Traversal)
- Binary Search Tree (BST)

---

## 🌳 Tree Operations
- Inorder, Preorder, Postorder Traversal
- Search
- Insertion
- Deletion

---

## ⚙️ Concepts Used
- Pointers
- Dynamic Memory Allocation (`malloc`, `free`)
- Recursion
- Modular Programming (Header + Source files)

---

## 📁 Project Structure

```
dsa-toolkit-c/
│
├── stack/
├── queue/
├── linkedlist/
├── tree/
│
├── main.c
└── README.md
```

## ▶️ How to Run

```bash
gcc main.c stack/*.c queue/*.c linkedlist/*.c tree/*.c -o main
./main
```
